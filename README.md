
# Curso de Lógica de Programação - Alura
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Este repositório contém as atividades desenvolvidas durante o Curso de Lógica de Programação da Alura + Oracle.

---

O curso abrangeu o uso de tecnologias fundamentais para o desenvolvimento de páginas web, incluindo JavaScript, HTML e CSS.

Foram implementados quatro projetos práticos para aplicar os conhecimentos adquiridos:
### [Jogo da Adivinhação - Diálogo](https://dialog-game.vercel.app/)
Este jogo, desenvolvido em JavaScript, utiliza conceitos básicos de programação. Baseia-se na tentativa de adivinhar um número gerado aleatoriamente pela máquina. Foram empregadas funções de diálogo nativas do JavaScript, como `alert()`, `confirm()` e `prompt()`.
- [Branch](https://github.com/joaoxmb/alura-logica-de-programacao/tree/dialog-game)
- [Produção](https://dialog-game.vercel.app/)

### [Jodo da Adivinhação - Input](https://input-game.vercel.app/)
Essa é a segunda versão do Jogo da Adivinhação - Diálogo, dessa vez foi utilizado conceitos mais complexos do JavaScript, como a manipulação do DOM, bem como a coleta de informações diretamente do HTML, utilizando a tag de entrada `<input>`. O jogo segue o mesmo princípio mas com uma melhora em sua lógica bem como o layout para o player.
- [Branch](https://github.com/joaoxmb/alura-logica-de-programacao/tree/input-game)
- [Produção](https://input-game.vercel.app/)

### [Portfólio v1](https://alura-html-css-portfolio-v1.vercel.app/)
Uma landing page de portfólio pessoal, durante esse projeto foi passado conceitos básicos de HTML e CSS, além de uma breve passada em como utilizar o Figma e seus benefícios para um projeto.
- [Branch](https://github.com/joaoxmb/alura-logica-de-programacao/tree/portfolio-v1)
- [Produção](https://alura-html-css-portfolio-v1.vercel.app/)

### [Portfólio v2](https://alura-portfolio-v2.vercel.app/)
Continuação do projeto Portfólio v1, esta versão inclui uma segunda página com um currículo pessoal. Além disso, foram aplicados conceitos de responsividade e flexbox para garantir a compatibilidade com diferentes dispositivos, incluindo desktops, tablets e dispositivos móveis.
 - [Branch](https://github.com/joaoxmb/alura-logica-de-programacao/tree/portfolio-v2) 
-  [Produção](https://alura-portfolio-v2.vercel.app/)
